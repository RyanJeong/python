## Python in VSCode
1. Install VSCode
2. Install extensions from the VSCode:
    1. `Python`
    2. `Python for VSCode`
    3. `Python Extension Pack`
    4. `Jupyter`
    5. `pylint`, `jupylint`

## Run Python Program in VSCode
1. Create a new file(.py)
2. Wirte codes and press `Ctrl + F5` for run the program
